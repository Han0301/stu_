---

## 一、先搞懂：MinMaxScaler 是什么？

**全称**：最小最大归一化  
**作用**：把任意数值范围的特征，压缩到 **[0, 1] 之间**

### 核心公式
$$
X_{归一化} = \frac{X - X_{min}}{X_{max} - X_{min}}
$$

| 原始值 | 归一化后 |
|------|---------|
| 最小值 | → 0 |
| 最大值 | → 1 |
| 其余值 | 均匀分布在 0~1 之间 |

---

## 二、为什么要用它？

1. 让**不同量级的特征**（Age：0~80，Fare：0~500）变成同一尺度  
2. **必须用于**：逻辑回归、SVM、神经网络、KNN 等模型  
3. **不用**：随机森林、XGBoost 等树模型  

---

## 三、核心语法（4 步固定流程）

### 1️⃣ 导入工具
```python
from sklearn.preprocessing import MinMaxScaler
```

### 2️⃣ 创建归一化器
```python
scaler = MinMaxScaler()
```
> 这只是**初始化**，还没计算任何数据

### 3️⃣ 训练 + 转换（一步到位）
```python
scaled_data = scaler.fit_transform(原始数值数据)
```

| 方法 | 作用 |
|----|----|
| `fit()` | 计算特征的 **最大值、最小值**（学习规则） |
| `transform()` | 根据规则，把数据缩放到 0~1（执行转换） |
| `fit_transform()` | **先学习 + 再转换** |

### 4️⃣ 覆盖原数据（常用）
```python
df['特征列'] = scaled_data
```

---

## 四、关键细节（新手必看）

### ✅ 必须传入 **二维数组**
`MinMaxScaler` 要求输入是 **二维格式（列向量）**

| 写法 | 是否正确 | 原因 |
|----|----|----|
| `df[['Age']]` | ✅ 正确 | 双层括号 → 二维 |
| `df['Age']` | ❌ 错误 | 单层括号 → 一维（会报错） |

---

### ✅ 默认范围是 [0, 1]，也可以自定义
```python
# 缩放到 [-1, 1]
scaler = MinMaxScaler(feature_range=(-1, 1))
```

---

### ✅ 只对 **数值特征** 使用

| 类型 | 是否使用 |
|----|----|
| Age、Fare、SibSp、Parch | ✅ 可以 |
| 独热编码后的 0/1 特征 | ❌ 不可以 |
| 分类特征 | ❌ 不可以 |

---

## 五、【你的泰坦尼克数据】实战代码

你的数值特征：
```
Age、Fare、SibSp、Parch
```

### ✅ 完整归一化代码（直接插在独热编码前）
```python
# ===================== MinMaxScaler 归一化数值特征 =====================
# 1. 导入工具
from sklearn.preprocessing import MinMaxScaler

# 2. 定义需要归一化的 数值列（分类特征不要动）
numeric_cols = ['Age', 'Fare', 'SibSp', 'Parch']

# 3. 创建归一化器
scaler = MinMaxScaler()

# 4. 批量归一化所有数值列（核心代码）
all_df[numeric_cols] = scaler.fit_transform(all_df[numeric_cols])

# 查看归一化结果（0~1之间）
print("✅ 归一化后 Age 范围：", all_df['Age'].min(), "~", all_df['Age'].max())
print("✅ 归一化后 Fare 范围：", all_df['Fare'].min(), "~", all_df['Fare'].max())
```

### ✅ 运行结果
```
✅ 归一化后 Age 范围： 0.0 ~ 1.0
✅ 归一化后 Fare 范围： 0.0 ~ 1.0
```

numeric_cols = ['Age', 'Fare', 'SibSp', 'Parch']

```python
scaler = MinMaxScaler()
```

```python
all_df[numeric_cols] = scaler.fit_transform(all_df[numeric_cols])
```
👉 工具学习这 4 列的 **最大/最小值**  
👉 自动把所有值缩放到 **0~1**

## 七、训练集 / 测试集 必知规则（进阶）

你是**合并 train + test 一起预处理**，所以直接用 `fit_transform` **完全没问题** ✅  
这是竞赛标准做法，可以避免数据泄露。

---

## 八、新手最常见的 2 个错误

### ❌ 错误 1：单层括号（一维数组报错）
```python
# 错误
all_df['Age'] = scaler.fit_transform(all_df['Age'])

# 正确
all_df['Age'] = scaler.fit_transform(all_df[['Age']])
```

---

### ❌ 错误 2：对分类特征归一化
```python
# 错误（独热特征是 0/1，不需要归一化）
all_df['Sex_male'] = scaler.fit_transform(all_df[['Sex_male']])
```

---

## 九、极简总结（必背）

1. **MinMaxScaler**：把数值缩放到 **0~1**
2. **4 步用法**：导入 → 创建 → fit_transform → 覆盖
3. **只用于**：数值特征（Age / Fare）
4. **必须双层括号**：`[['列名']]`
5. **适用模型**：线性模型、神经网络、SVM
6. **树模型（随机森林）**：**完全不用归一化**
