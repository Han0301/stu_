## 二、完整代码：表格 → 神经网络输入（直接复制运行）

```python
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
import numpy as np

# ===================== 步骤1：处理你的数据，构建合法输入 =====================
# 1. 复制你的数据（避免修改原数据）
df = all_df.copy()

# 2. 【必须】删除无用列：PassengerId
df = df.drop(columns=["PassengerId"])

# 3. 【必须】bool类型转 0/1 数值（神经网络不识别布尔值）
df = df.astype('float32')  # 一键把所有列转成神经网络支持的浮点型

# 4. 拆分：训练集（有标签） / 测试集（无标签）
train_df = df[df['Survived'].notna()]  # 891行，有标签
test_df = df[df['Survived'].isna()]   # 418行，无标签

# 5. 【核心】分离 输入特征(X) 和 标签(y)
# 训练集输入：所有列 排除 Survived
X_train = train_df.drop(columns=["Survived"]).values  
# 训练集标签：只取 Survived
y_train = train_df["Survived"].values  

# 测试集输入（用于最终预测）
X_test = test_df.drop(columns=["Survived"]).values  

# ===================== 查看输入结果（关键验证） =====================
print("训练集输入形状 (样本数, 特征数)：", X_train.shape)  # (891, 23) → 23个输入特征
print("训练集标签形状：", y_train.shape)                # (891,)
print("测试集输入形状：", X_test.shape)                 # (418, 23)
print("输入数据类型：", X_train.dtype)                  # float32 (模型支持)
```

## 三、关键步骤详细解释

### 1. `df = df.astype('float32')`
把你的 `bool` 列（`Sex_male、Cabin_B`等）**自动转为 0.0 / 1.0**
- True → 1.0
- False → 0.0
这是神经网络**强制要求**的输入类型。

### 2. `.values`
把 Pandas 表格 **转为 Numpy 数组**
- 这是 TensorFlow/Keras 唯一支持的输入格式
- 你的 23 个特征，会变成模型的**输入维度**

### 3. 最终输入结构
模型输入是一个 **二维数组**：
```
(891, 23)
891 个乘客，每个乘客有 23 个输入特征
```
这 23 个特征就是：
`Age、Fare、Sex_male、Embarked_Q、Embarked_S、Cabin_B、Cabin_C...Pclass_3`

## 四、搭建【线性神经网络】（适配你的输入）

你的输入维度是 **23**，直接搭建最简单的线性网络：

```python
# ===================== 步骤2：搭建线性神经网络 =====================
model = Sequential([
    # 输入层：input_dim=23 (你的特征数量)，线性激活
    Dense(16, activation='linear', input_dim=X_train.shape[1]),
    # 隐藏层：纯线性变换
    Dense(8, activation='linear'),
    # 输出层：二分类，输出0~1的生存概率
    Dense(1, activation='sigmoid')
])

# 编译模型（线性网络标配）
model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)

# 打印模型结构（确认输入维度=23）
model.summary()

# ===================== 步骤3：训练模型 =====================
print("\n开始训练线性神经网络...")
model.fit(
    X_train, y_train,
    epochs=50,      # 训练50轮
    batch_size=8,   # 每次喂8个样本
    validation_split=0.2,  # 用20%数据做验证
    verbose=1
)
```
