## 一、核心理论（贴合你的任务）

### 1. 两个算法的分工（你的ROI图像分类场景）

| 算法 | 核心作用 | 解决问题 | 适用你的任务 |
|------|----------|----------|--------------|
| **CoreSet (K-Center Greedy)** | 从海量ROI图片中，挑选**覆盖所有场景、无冗余、最具代表性**的子集 | 海量数据算力不够、重复样本浪费训练资源 | 保证筛选的样本覆盖所有类型的ROI特征（多样性） |
| **Margin-based Sampling** | 从核心集中，挑选**模型最纠结、决策边界附近**的困难样本 | 模型对易混淆样本预测不准、学习效率低 | 让模型专攻最容易出错的样本（困难性） |

### 2. 组合逻辑（工业界最优策略：CoreSet → Margin）
1. 先用CoreSet做**初筛**：剔除冗余，保留全局代表性；
2. 再用Margin做**精筛**：从代表性样本中挑最难的样本；
3. 最终用**1%~10%的精简数据**，达到全量数据95%以上的训练效果。

### 3. 核心原则（杜绝特征偏差）
**绝对不使用ResNet/CLIP等通用模型提取特征**，必须用**你自己训练的YOLO11 ROI模型**提取**任务专属特征**，保证特征空间100%对齐。

## 三、全流程落地步骤（超详细，一步一操作）

### 步骤1：配置模型 + 加载权重（适配你的YOLO11模型）
**作用**：加载训练好的模型，切换到推理模式，为特征提取和Margin计算做准备。
- 关键：参数必须和训练时**完全一致**（模型尺寸、分类数、ROI数量）。

### 步骤2：提取「任务专属特征」（核心！零偏差）
**作用**：将每张64×64 ROI图片，转换为模型训练时使用的**特征向量**（特征空间坐标），用于CoreSet距离计算。
- 特征提取位置：你的模型 `spatial_attention` 输出层（经过Backbone+Neck+注意力融合，最核心的语义特征）；
- 输出：固定维度特征向量（n:32维 / s:64维 / l:128维）。

### 步骤3：CoreSet初选 - K-Center Greedy 筛选代表性样本
**原理**：贪心算法，每次选择**距离已选样本最远**的图片加入核心集，保证覆盖所有场景，无冗余。
**输入**：所有图片的专属特征矩阵；
**输出**：最具代表性的样本索引（全量数据的5%~10%）。

### 步骤4：Margin-based精筛 - 筛选模型最困难的样本
**原理**：
1. 用你的模型预测核心集样本的分类概率；
2. 计算 `Margin = 最高预测概率 - 次高预测概率`；
3. 选择**Margin最小**的样本（模型最混淆、最难区分）。
**输入**：核心集样本的模型预测概率；
**输出**：困难样本索引（核心集的1%~5%）。

### 步骤5：构建最终精简训练集
合并CoreSet+Margin筛选后的样本，得到**精兵强将**数据集。

### 步骤6：用精简数据集训练/微调模型
用最终筛选的少量样本训练模型，算力节省90%以上，精度不下降。

### 步骤7：迭代优化（进阶，工业界标准）
1. 用精简数据训完一轮后，得到新权重；
2. 用新权重重新提取特征 → 重新筛选样本；
3. 迭代1~2轮，效果逼近全量训练。

## 四、完整整合代码（你的模型专用，复制即跑）

```python
import torch
import numpy as np
from torchvision import transforms
from PIL import Image
from sklearn.metrics import pairwise_distances
# 导入你的自定义模型
from model import YOLO11ROIClassifier

# ===================== 【0】全局配置（和训练完全一致） =====================
DEVICE = "cuda" if torch.cuda.is_available() else "cpu"
MODEL_SIZE = "n"        # 你的模型尺寸 n/s/l
NUM_CLASSES = 2         # 分类数
NUM_ROI = 12            # 固定12
ROI_SIZE = 64           # 固定64
ATTN_WEIGHT = 0.15
CHECKPOINT_PATH = "best_ckpt.pth"  # 你的权重路径
IMAGE_PATHS = ["roi_imgs/1.jpg", "roi_imgs/2.jpg", ...]  # 海量ROI图片路径
CORESET_RATIO = 0.1     # CoreSet筛选比例：10%
MARGIN_RATIO = 0.1      # Margin筛选比例：核心集的10%

# ===================== 【1】加载模型 =====================
model = YOLO11ROIClassifier(
    model_size=MODEL_SIZE, num_roi=NUM_ROI, num_classes=NUM_CLASSES,
    roi_size=ROI_SIZE, atten_weight=ATTN_WEIGHT
)
checkpoint = torch.load(CHECKPOINT_PATH, map_location=DEVICE)
model.load_state_dict(checkpoint)
model.to(DEVICE)
model.eval()

# 图片预处理（必须和训练一致！）
transform = transforms.Compose([
    transforms.Resize((ROI_SIZE, ROI_SIZE)),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

# ===================== 【2】单张ROI特征提取（你的模型专属） =====================
@torch.no_grad()
def extract_feature(img_path):
    img = Image.open(img_path).convert("RGB")
    img_tensor = transform(img).to(DEVICE).unsqueeze(0).unsqueeze(0)  # [1,1,3,64,64]
    
    # 截断模型，提取核心特征（不进分类头）
    B, N = img_tensor.shape[0], img_tensor.shape[1]
    roi_flat = img_tensor.reshape(-1, 3, ROI_SIZE, ROI_SIZE)
    feat_backbone = model.backbone(roi_flat)
    feat_neck = model.neck(feat_backbone)
    feat_attn = feat_neck.reshape(B, N, -1)
    feat, _ = model.spatial_attention(feat_attn)
    
    return feat.squeeze().cpu().numpy()

# ===================== 【3】批量提取所有特征 =====================
print("正在批量提取特征...")
all_features = np.array([extract_feature(path) for path in IMAGE_PATHS])
TOTAL_NUM = len(all_features)
CORESET_NUM = int(TOTAL_NUM * CORESET_RATIO)
MARGIN_NUM = int(CORESET_NUM * MARGIN_RATIO)

# ===================== 【4】CoreSet初选：K-Center Greedy =====================
def k_center_greedy(features, k):
    selected = [np.random.randint(len(features))]
    for _ in range(k-1):
        dist = pairwise_distances(features, features[selected]).min(axis=1)
        selected.append(np.argmax(dist))
    return selected

coreset_indices = k_center_greedy(all_features, CORESET_NUM)
print(f"CoreSet筛选完成：选中{len(coreset_indices)}个代表性样本")

# ===================== 【5】Margin精筛：挑选模型最困难的样本 =====================
@torch.no_grad()
def get_model_probs(img_paths):
    """用你的模型预测样本概率"""
    probs = []
    for path in img_paths:
        img = Image.open(path).convert("RGB")
        tensor = transform(img).to(DEVICE).unsqueeze(0).unsqueeze(0)
        logits = model(tensor)  # [1,1,2]
        prob = torch.softmax(logits, dim=-1).squeeze().cpu().numpy()
        probs.append(prob)
    return np.array(probs)

# 预测核心集样本概率
coreset_img_paths = [IMAGE_PATHS[i] for i in coreset_indices]
coreset_probs = get_model_probs(coreset_img_paths)

# 计算Margin并筛选
def margin_sampling(probs, n_sample):
    top2 = np.partition(probs, -2, axis=1)[:, -2:]
    margins = top2[:, 1] - top2[:, 0]
    return np.argsort(margins)[:n_sample]

hard_indices = margin_sampling(coreset_probs, MARGIN_NUM)
print(f"Margin筛选完成：选中{len(hard_indices)}个困难样本")

# ===================== 【6】构建最终训练集 =====================
final_train_indices = [coreset_indices[i] for i in hard_indices]
final_train_images = [IMAGE_PATHS[i] for i in final_train_indices]
print(f"最终精简训练集：共{len(final_train_images)}张图片")

# ===================== 【7】用精简数据集训练模型 =====================
# 此处替换为你的训练代码，仅使用 final_train_images 训练
```

## 六、避坑指南（必看！保证效果）
1. **特征绝对不能用通用模型**
   必须用你的YOLO11模型提取特征，否则特征空间偏差，筛选无效。
2. **预处理必须和训练完全一致**
   Resize、Normalize参数一丝不差，否则特征提取错误。
3. **输入必须是64×64 ROI小图**
   不要输入大图，你的模型是为ROI设计的。
4. **Margin必须用你的模型预测**
   不能用其他模型计算概率，否则困难样本筛选错误。
5. **不要单独用Margin**
   单独用Margin会选大量雷同样本，必须先做CoreSet去重。
