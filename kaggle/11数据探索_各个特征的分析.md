### 第一阶段：数据探索（Data Exploration）
**定义**：数据探索是使用统计方法和可视化技术，对数据集的结构、质量（缺失值、异常值）和分布进行初步审查的过程。它是发现数据规律的“侦察兵”。

#### 1. 数据加载与全景扫描
**操作**：使用 `pandas.read_csv` 读取数据，并使用 `.info()` 和 `.describe()` 进行元数据审查。

```python
import pandas as pd
import numpy as np

# 读取 CSV 文件
# 语法解释：pd.read_csv() 会将表格数据转换为 DataFrame 结构
train = pd.read_csv('train.csv')

# 查看数据结构
# 语法解释：.info() 显示列名、非空值数量和数据类型
print(train.info())
```
**输出解读**：
- **RangeIndex**: 891 entries，表示 891 行数据。
- **Non-Null Count**: `Age` 只有 714 个非空值，说明缺失了 177 个。
- **Dtype**: `object` 通常是字符串（如 Name, Sex），`int64/float64` 是数值。

**生动解释**：
这就像**医生接诊前的体检**。`.info()` 是看**血常规**，检查身体（数据集）里有多少器官（列），哪些器官缺血（缺失值），以及器官的功能状态（数据类型）。

```python
# 查看统计摘要
# 语法解释：.describe() 只对数值列生效，计算均值、标准差、最值等
print(train.describe())
```
**生动解释**：
`.describe()` 是**量体温和血压**。`mean` 是平均体温，`std` 是波动幅度。如果你看到 `Fare` 的最大值是 512，而 75% 分位数才 31，说明有个**超级富豪**把平均值拉高了，这就是异常值。

### 第二阶段：变量分析（Variable Analysis）
**定义**：根据变量的测量尺度（Scale），将其分类并分析其与目标变量（Target Variable）的关系。

#### 1. 单变量分析（Univariate Analysis）
**定义**：关注单个变量的分布情况。

| 变量类型 | 分析方法 | 核心关注点 |
| :--- | :--- | :--- |
| **分类变量 (Categorical)** | 频次统计 | 各类别的占比是否失衡 |
| **数值变量 (Numerical)** | 分布检验 | 是否正态分布，是否存在长尾 |

**代码示例：分类变量分析**
```python
# 查看 Survived 的分布
# 语法解释：.value_counts() 统计每个唯一值的出现次数
survival_counts = train['Survived'].value_counts()
print(survival_counts)
# 输出：0    549, 1    342
```
**生动解释**：
这是**清点战场伤亡**。0（死亡）有 549 人，1（存活）有 342 人。说明这是一个**不平衡数据集**，死的人比活的多。

#### 2. 双变量分析（Bivariate Analysis）
**定义**：分析特征（X）与目标变量（Y）之间的关系。

**场景**：探究“性别（Sex）”对“存活（Survived）”的影响。

```python
# 计算不同性别的存活率
# 语法解释：groupby('Sex') 按性别分组，['Survived'].mean() 计算每组的均值（即存活率）
survival_by_sex = train.groupby('Sex')['Survived'].mean()
print(survival_by_sex)
```
**生动解释**：
这就像**分班看成绩**。结果会显示女性的存活率远高于男性。在泰坦尼克号上，**“女士优先”**不是口号，是生存法则。

### 第三阶段：可视化分析（Visualization with Matplotlib/Seaborn）
**定义**：利用视觉通道（长度、位置、颜色）将数据映射为图形，利用人类视觉系统的高带宽特性发现隐藏模式。

#### 1. 分类变量 vs 目标（条形图）
**场景**：可视化“舱位等级（Pclass）”与“存活率”的关系。

```python
import seaborn as sns
import matplotlib.pyplot as plt

plt.figure(figsize=(8, 5)) # 设置画布大小
# 语法解释：sns.barplot 绘制条形图
# x='Pclass': 横轴是舱位等级
# y='Survived': 纵轴是存活率的均值
# data=train: 数据源
sns.barplot(x='Pclass', y='Survived', data=train, palette='Reds')
plt.title('Survival Rate by Passenger Class')
plt.show()
```
**逐行逻辑**：
1. `plt.figure()`: 准备一张画纸。
2. `sns.barplot()`: 告诉 Seaborn，我要画柱状图。它会自动计算每个 Pclass 对应的 Survived 平均值。
3. `palette='Reds'`: 使用红色系，暗示危险与生存。

**生动解释**：
这张图展示了**阶级的鸿沟**。一等舱（1）高高在上，三等舱（3）趴在地上。这直观地告诉我们：**钱能买通救生艇的门票**。

#### 2. 数值变量分布（直方图与密度图）
**场景**：查看“年龄（Age）”的分布，并对比幸存者与遇难者。

```python
plt.figure(figsize=(12, 5))

# 左图：整体年龄分布
plt.subplot(1, 2, 1) # 1行2列的第1个位置
# 语法解释：histplot 直方图，kde=True 叠加核密度估计曲线（平滑后的分布）
sns.histplot(data=train, x='Age', kde=True, color='skyblue')
plt.title('Overall Age Distribution')

# 右图：存活与死亡的对比
plt.subplot(1, 2, 2)
# 语法解释：hue='Survived' 按存活状态着色，0为蓝，1为橙
sns.kdeplot(data=train, x='Age', hue='Survived', fill=True, common_norm=False)
plt.title('Age vs Survival')
plt.show()
```
**生动解释**：
*   **左图**：像一座山，小孩多，老人少。
*   **右图**：这是**生命的岔路口**。在 0-10 岁区间，橙色（幸存者）的面积远大于蓝色（遇难者）。这证明了**“妇女儿童优先”**的执行力度。而在老年区，两者几乎重叠，说明上帝在这里关上了门。

### 第四阶段：建模特征关联（Feature Association）
**定义**：量化特征之间的线性或非线性依赖关系，用于特征选择（Feature Selection）和降维（Dimensionality Reduction）。

#### 1. 相关性热力图（Correlation Heatmap）
**定义**：使用皮尔逊相关系数（Pearson Correlation Coefficient）衡量两个连续变量之间的**线性相关程度**，取值范围 [-1, 1]。

```python
# 筛选数值型特征
numeric_df = train.select_dtypes(include=np.number)

plt.figure(figsize=(10, 8))
# 语法解释：.corr() 计算相关系数矩阵
corr_matrix = numeric_df.corr()

# 语法解释：sns.heatmap 绘制热力图
# annot=True: 在格子里标注具体数值
# cmap='coolwarm': 冷暖色系，红正蓝负
# center=0: 颜色中心点为0
sns.heatmap(corr_matrix, annot=True, cmap='coolwarm', center=0)
plt.title('Feature Correlation Matrix')
plt.show()
```
**逐行逻辑**：
1. `select_dtypes(include=np.number)`: 只保留数字列，因为相关系数只能算数字。
2. `.corr()`: 生成一个矩阵，坐标 (i, j) 的值代表 i 和 j 的相关系数。
3. `sns.heatmap()`: 把矩阵涂色。

**解读与生动解释**：
*   **Survived 行**：看它与谁的颜色最深。
    *   `Survived` vs `Fare`: 浅红色（正相关）。票越贵，越容易活。
    *   `Survived` vs `Pclass`: 深蓝色（负相关）。舱位等级数字越大（越穷），死得越快。
*   **特征间**：`SibSp` 和 `Parch` 呈红色，说明带兄弟姐妹的人通常也带父母，信息重叠（多重共线性）。

#### 2. 配对图（Pair Plot）
**定义**：一种网格图，对角线上是单变量分布，非对角线上是两个变量的散点图。

```python
# 选取关键特征进行配对分析
sns.pairplot(train[['Survived', 'Age', 'Fare', 'Pclass']], hue='Survived')
plt.show()
```
**生动解释**：
这就像把数据放在**显微镜下旋转观察**。你会看到 `Fare` 和 `Pclass` 之间有明显的线性分层（越贵舱位越低），但 `Age` 和 `Fare` 之间则是一团乱麻，说明它们是独立的特征。
